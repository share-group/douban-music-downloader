# 豆瓣音乐下载器

#### 使用方法
```
1.把 playlist.json.example 复制一份，命名 playlist.json
2.复制 https://music.douban.com/artists/player/ 页面的变量 __bootstrap_data 的内容到playlist.json，即可下载
3.cd /path/to/douban-music-downloader
4.yarn
5.yarn start
6.完成，听歌去！
```
